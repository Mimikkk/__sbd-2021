# Projekt zaliczeniowy z sbd

Aplikacja symulująca działanie systemu administracyjnego w firmie zarządzającej obiektem sportowym.

## Wymagania aplikacji

- Node 17^
- npm 8^

## Sposób uruchumienia

> sklonuj repozytorium

> Pobranie zależności
>> npm install

> Zbudowanie projektu
>> npm build

> Uruchomienia
>> npm start
